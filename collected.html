Assumptions (auto-detected per bill): uses these fields if present: amount (collection), sellingAmount / selling, productAmount / product, gstPercent / gst or gstAmount. If missing, it falls back to reasonable defaults and shows a note.

How Profit Excluding GST is Calculated

Formula
Profit = (Selling Excl. GST) − (Product Cost Excl. GST)
If you store GST as percent: excl = incl / (1 + gst%/100)
If you store GST amount: excl = incl − gstAmount
Field Mapping (auto)

Customer: customer / customerName / name

Tip
For best accuracy, store per-bill: sellingAmount, productAmount, and either gstPercent or gstAmount.
